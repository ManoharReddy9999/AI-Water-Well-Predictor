# AI-Enabled Water Well Predictor

This project uses Machine Learning to predict groundwater availability and optimize well drilling decisions.

## Features

* Predicts well suitability
* Estimates depth and discharge
* Suggests drilling method
* Predicts water quality

## Tech Stack

* Python
* Scikit-learn
* XGBoost
* Pandas
* Streamlit

## Goal

To support sustainable water resource management using AI.
